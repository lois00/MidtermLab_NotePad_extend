# MidtermLab_NotePad_Extend
## 项目说明：本实验是基于谷歌SDK NotePad应用做的功能扩展。<br>
## 功能列表：<br>
### 1.初始功能：<br>
列出笔记条目、添加笔记、保存笔记、删除笔记、修改笔记的标题、
粘贴、编辑笔记内容与撤销编辑操作<br>
### 2.扩展功能：<br>
笔记条目增加时间戳显示、根据标题查询笔记、<br>
UI美化、笔记排序、导出笔记、更改记事本的背景<br>
